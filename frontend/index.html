📁 Huấn luyện từ File Import

Upload file CSV hoặc Excel chứa dữ liệu huấn luyện. File cần có các cột bắt buộc:

age gender marital_status monthly_income living_expenses current_debt asset_value dependents housing_status loan_amount purpose loan_term_months package_id is_on_time_payment

📌 Giá trị hợp lệ:

gender: male, female
marital_status: single, married, divorced
housing_status: own, rent, stay_with_parents
purpose: bds, o_to, tieu_dung, kinh_doanh
package_id: 1-10 (theo gói vay hiện có)
is_on_time_payment: 0 hoặc 1

📊 Yêu cầu:

• Tối thiểu 50 hàng dữ liệu hợp lệ
• Ít nhất 2 loại package_id khác nhau
• File CSV mã hóa UTF-8
• Excel: sheet đầu tiên được sử dụng
